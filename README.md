## 📚 Содержание репозитория

В ходе обучения проект развивался от создания простых объектов до сложной архитектуры на основе интерфейсов:

### [ЛР-1: Сущность (Entity)](./src/lab01/README.md)
*   **Тема:** Проектирование базового класса `Player`.
*   **Ключевые навыки:** Инкапсуляция, свойства (`@property`), валидация данных и магические методы (`__str__`, `__repr__`).

### [ЛР-2: Коллекция объектов (Container)](./src/lab02/README.md)
*   **Тема:** Создание контейнерного класса `PlayerCollection`.
*   **Ключевые навыки:** Управление группой объектов, итерация (`__iter__`), индексация (`__getitem__`) и логическая фильтрация.

### [ЛР-3: Наследование и иерархия (Hierarchy)](./src/lab03/README.md)
*   **Тема:** Построение иерархии классов `Warrior` и `Mage`.
*   **Ключевые навыки:** Наследование, использование `super()`, переопределение методов и базовый полиморфизм.

### [ЛР-4: Интерфейсы и абстрактные классы (ABC)](./src/lab04/README.md)
*   **Тема:** Проектирование архитектуры на основе контрактов.
*   **Ключевые навыки:** Абстрактные методы (`@abstractmethod`), множественная реализация интерфейсов (`IPrintable`, `ICombatant`) и полиморфизм интерфейсов.
